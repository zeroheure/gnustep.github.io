The GNUstep Framework And Applications

GNUstep is a framework that follows Apple's Cocoa APIs. Portable and mature it is suited for desktop and server apps. Users will find mature applications and desktop projects to install. While developers will find tools and libraries for writing portable apps for macOS, Windows, Linux, BSD, iOS, Android, on x86/ARM and others.

Well conceived user experience

Some really good applications have been developped with the GNUstep framework. All GNUstep applications are well integrated and use native look and feel of operating systems.

Ergonomic applications

All GNUstep applications provide a consistent, ergonomic, user experience with a high quality GUI, because GNUstep is build on giants shoulder, thanks to Steve Jobs and his team of conceptors at Apple and Next Computer. The superior design and consistent interface comes directly from NextStep and Apple specifications.

Green applications

Being based of the original NextStep framework our apps use low resources. All of them are fast and full of features even on an old computers.

Available on your Operating System

GNUstep applications can be installed on Windows, macOS, Linux, the BSD family and other Unixes. Some apps are also available for Android and iOS.

Full desktop

Some free software desktop projects have been started, mainly for Linux and BSD systems, using GNUstep as a solid foundation. The completion of a desktop environment is done by other projects.

  • NextSpace try to rebuild the user experience of NeXTSTEP famous desktop. With an accurate, clear, grayish and boring interface that helps, not hinder, to get your job done.
  • Étoilé looks more like macOS. It is designed around the things people do with computers: create, collaborate, and learn. It should be portable to most operating systems.
  • The GNUstep Application Project provide applications that users can install inside WindowMaker, with GNUstep's GWorkspace as a desktop manager. This is a bit tough but provide a decent desktop for advanced users on old computers.
  • Other projects have been started

Developer experience

GNUstep provides an environment to easily develop advanced GUI desktop applications as well as server applications.

Code once, compile everywhere

GNUstep's core framework provides open source version of the Apple's Cocoa APIs and tools for as many platforms as possible. GNUstep seeks to be source code compatible with Cocoa, it can thus be used to develop and build cross-platform applications between Macintosh (Cocoa), Unix (Solaris) and Unix-like (GNU/Linux and GNU/Hurd, NetBSD, OpenBSD, FreeBSD) platforms as well as Windows

Faster coding with developers tools

GNUstep provides a robust implementation of the AppKit and Foundation libraries from Apple macOS and NextStep. The GNUstep team has also build advanced development tools available on every OS. These include the polished interface designer Gorm (very similar to InterfaceBuilder on macOS) and the IDE ProjectCenter (like ProjectBuilder / Xcode on Apple's macOS). These tools provides interfaces to compilers thas support several programming languages.

Develop in your favorite language

GNUstep's main development language is Objective-C. But our softwares are not limited to Objective-C and developers can quickly design and build cross-platform applications in Objective-C, Java, Ruby, Swift and C language. Gtk bindings
